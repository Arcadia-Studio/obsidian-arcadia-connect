# Arcadia Connect

A personal CRM for Obsidian: link people across your notes with @-mentions, log interactions, and manage relationships from a dedicated People sidebar.

## Features

### Free
- @-mention autocomplete in the editor (CodeMirror 6 extension)
- Person note creation from a command or the People Panel
- People sidebar with a live, searchable list of all contacts
- Manual interaction logging on person notes
- Hover profile cards when reading @-mentions
- Real-time index updates on file create, rename, and delete

### Premium
- Interaction timeline view per person
- Birthday reminders and upcoming-date alerts
- Follow-up reminders with configurable intervals
- Group and organization views
- CSV import and export for bulk contact management
- Expanded profile card sidebar with tags, notes, and relationship metadata
- Get Premium at [arcadia-studio.lemonsqueezy.com](https://arcadia-studio.lemonsqueezy.com)

## Installation

1. Open Obsidian Settings
2. Go to Community Plugins and disable Safe Mode
3. Click Browse and search for "Arcadia Connect"
4. Install and enable the plugin

## Manual Installation

1. Download the latest release from [GitHub Releases](https://github.com/Arcadia-Studio/obsidian-arcadia-connect/releases)
2. Extract to your vault's `.obsidian/plugins/arcadia-connect/` folder
3. Reload Obsidian and enable the plugin

## Usage

Type `@` in any note to trigger the person autocomplete. Select a name to insert a mention link. Mentioned people appear automatically in the People Panel sidebar.

Open the People Panel via the ribbon icon (users), the command palette ("Open People Panel"), or the keyboard shortcut you assign in Settings.

To create a person note directly, use the "Create Person Note" command from the command palette.
